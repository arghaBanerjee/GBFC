# Local Football Club Site (Streamlit)

A simple Streamlit-based website for a local football club with:

- Home / landing page with football-themed hero section
- Email + password signup/login (SQLite backend)
- Events page with past and upcoming events, likes, interest, and comments
- Practice page with Thursday practices and availability voting
- About Us page with committee and squad overview

## Getting started

1. Create and activate a virtual environment (optional but recommended).
2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the Streamlit app:

```bash
streamlit run app.py
```

4. Open the URL shown in the terminal (usually http://localhost:8501).

You can customise sample content (club name, images, events) directly in `app.py`.
